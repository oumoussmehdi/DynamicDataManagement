Please find in this folder:
1. Project_Report.pdf : the report about the project.

2. wsproject_oumouss.jar : is a runnable jar 
(for execution : java -jar wsproject_oumouss.jar)

3. OUMOUSS_WSProject.zip : contains source code for ws-execution,
which is the implementation of our project.

4. WWW : contains the web services definition
please in the setting class in the implementation project, 
define the path to the WWW folder 

5. sqlCmds: a file that contains the commands used to create the DB.
in your mysql database, you can run the next command : 
(mysql> source /path_to_DCM_folder/DCM/Project/sqlCmds)
and this will create the database and its tables.
